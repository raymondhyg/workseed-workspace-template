Generated by: `scripts/utils/create_workspace_deployment.py`

Core release: `ws-core-v0.3.0`

This folder is the built-in feedback interface for Workspace self-repair,
local judgment, and sanitized Core signals.

## Default Flow

1. Self-repair locally when the issue belongs to Workspace facts, materials, or decisions.
2. Record the decision locally when it affects future work.
3. Prepare a sanitized Core signal only when the issue may affect other Workspaces or the Template.
4. Keep private facts, raw assets, account state, private links, adapter details, and external task contents out of Core feedback.
